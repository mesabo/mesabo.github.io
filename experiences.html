製品 ＆ エンジニアリング

  • CVFlow（cvflow.app） —  AI 履歴書ビルダー: Next.js / React フロントエンド、GCP Cloud Run 上の FastAPI バックエンド、Firestore + Firebase Auth、Google Gemini による履歴書解析・ATS スコアリング・カバーレター・面接対策、Stripe 課金、20 言語対応。リモート国際協業で構築。

  • AIReco  —  AI 会議アシスタント アプリ（Flutter、GCP）: オンデバイスのファインチューニング済み sLM と Gemini / GPT フォールバックにより会議を録音・文字起こしし、20 言語にわたる要約・キーワード・翻訳を生成。

  • SensoryHub  —  産業向け IoT 予知保全 SaaS（チーム）: ESP32 エッジ機器と Raspberry Pi ゲートウェイが MQTT 経由で Python FastAPI マイクロサービス（TimescaleDB、Redis）へストリーミング、ONNX で提供する ML 異常検知、React ダッシュボード、Kubernetes Terraform 上で稼働。

  • EKKOU プラットフォーム  —  専用 GPU バックエンドでのモデル提供を備えるフルスタック AI プラットフォーム。App Store および Google Play で公開中。

  • 専用 GPU サーバのデプロイ  —  一気通貫の所有: ハードウェア立ち上げ、CUDA／ドライバ インストール、コンテナ／CI 構築、sLM／LLM 学習ジョブ、推論デプロイ、監視。
