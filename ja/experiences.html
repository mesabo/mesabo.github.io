フルスタック ソフトウェア エンジニアリング

本番品質の Web（React Node.js Flask、REST および GraphQL、MongoDB）およびモバイル（Flutter Dart）。40 の公開リポジトリは github.com/mesabo に。

スタックと主要プロジェクト
  • Web: React フロントエンドが Flask または Node.js GraphQL バックエンドを利用; データ層は MongoDB および PostgreSQL。
  • モバイル: Flutter / Dart アプリ — crudium ⭐（Flutter スキャフォールディング ライブラリ）、chronohouse（時間管理）、progistack_mobile（生産性スタック）、flutter_graphql_io（GraphQL 統合）。
  • DevOps: Docker ベース開発フロー（debug_with_docker）。
