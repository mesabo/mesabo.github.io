Ingénieur Full-stack Ingénieur IA / Backend Chercheur

Je suis un ingénieur logiciel qui conçoit des applications web et mobiles full-stack et des systèmes IA backend de bout en bout  —  de l'interface aux API REST GraphQL, aux bases de données, au cloud et au CI/CD. Avec plus de 5 ans d'expérience dans les startups, l'industrie et la recherche, je livre des produits et les prends en charge du prototype à la production.

J'ai récemment livré CVFlow (cvflow.app, une plateforme de CV par IA), construit AIReco (un assistant de réunion IA), et je contribue à SensoryHub, une plateforme industrielle de maintenance prédictive IoT. Je suis également doctorant au YuLab, Université Hosei, où je mène des recherches sur des méthodes IA sLM LLM et IoT pour les séries temporelles et les réseaux électriques intelligents.

Axes d'ingénierie

  • Web full-stack ( React / Next.js) et mobile ( Flutter / Dart) — natif Android / iOS ( Swift, Java)
  • Services backend en Python ( Flask, FastAPI) et Node.js, avec REST et GraphQL
  • Couches de données: MongoDB · PostgreSQL · MySQL · Firebase / Firestore
  • Cloud: Google Cloud · conteneurs avec Docker · CI avec GitHub Actions
  • Intégration IA: API LLM (Gemini, GPT, Claude), sLM embarqués, service de modèles sur serveurs GPU
  • Applications LLM: LangChain / LangGraph / LangSmith, pipelines LCEL, RAG (FAISS, ChromaDB, RAGAS), et orchestration d'agents

Produits sélectionnés

  • CVFlow (cvflow.app)  —  plateforme de CV par IA: frontend Next.js / React, backend FastAPI sur GCP, score ATS et générateurs propulsés par Gemini, Stripe, interface en 20 langues. Construit en collaboration internationale à distance.

  • AIReco  —  assistant de réunion IA (Flutter, GCP): enregistrement, transcription et analyses IA en 20 langues; sLM affiné en local avec Gemini / GPT.

  • LangChain Production Stack (github.commesabolangchain-production-stack)  —  quatre systèmes LLM open-source de production: recherche sémantique, RAG, agents à état, et LLMOps full-stack sur LangChain LangGraph, FastAPI, et GCP avec CICD GitHub Actions.

  • SensoryHub  —  SaaS industriel de maintenance prédictive IoT (en équipe): dispositifs en périphérie → passerelles MQTT → microservices FastAPI → détection d'anomalies ML → tableaux de bord React, sur Kubernetes / Terraform.

  • EKKOU, Babi Gas, OKOHI  —  applications Flutter multiplateformes livrées avec Anohou Solutions.

  • IoT et intelligence en périphérie pour les énergies réparties

  • Réseaux de neurones sur graphes appliqués aux topologies électriques
