フルスタック エンジニア AI / バックエンド エンジニア 研究者

私は、フルスタックの Web・モバイル アプリケーションや AI／バックエンド システムを一気通貫で構築するソフトウェア エンジニアです。UI から REST／GraphQL API、データベース、クラウド、CI/CD まで担当します。スタートアップ・産業・研究にわたる 5 年以上の経験を活かし、プロトタイプから本番まで製品を所有し提供しています。

直近では CVFlow（cvflow.app、AI 履歴書プラットフォーム）を提供し、AIReco（AI 会議アシスタント）を構築、産業向け IoT 予知保全プラットフォーム SensoryHub にも貢献しています。また、法政大学 YuLab の博士課程に在籍し、時系列およびスマートグリッド システム向けの AI／sLM／LLM と IoT 手法を研究しています。

エンジニアリング分野

  • フルスタック Web（ React / Next.js）およびモバイル（ Flutter / Dart）— ネイティブ Android / iOS（ Swift、 Java）
  • Python（ Flask、 FastAPI）および Node.js によるバックエンド サービス、REST および GraphQL
  • データ層: MongoDB · PostgreSQL · MySQL · Firebase / Firestore
  • クラウド: Google Cloud · コンテナ Docker · CI GitHub Actions
  • AI 連携: LLM API（Gemini、GPT、Claude）、オンデバイス sLM、GPU サーバでのモデル提供

主な製品

  • CVFlow（cvflow.app） —  AI 履歴書プラットフォーム: Next.js / React フロントエンド、GCP 上の FastAPI バックエンド、Gemini による ATS スコアリングと各種ジェネレータ、Stripe、20 言語 UI。リモート国際協業で構築。

  • AIReco  —  AI 会議アシスタント（Flutter、GCP）: 録音・文字起こし・AI インサイトを 20 言語で提供。Gemini / GPT と組み合わせたオンデバイスのファインチューニング済み sLM。

  • SensoryHub  —  産業向け IoT 予知保全 SaaS（チーム）: エッジ機器 → MQTT ゲートウェイ → FastAPI マイクロサービス → ML 異常検知 → React ダッシュボード、Kubernetes / Terraform 上で稼働。

  • EKKOU、Babi Gas、OKOHI  —  Anohou Solutions で提供したクロスプラットフォーム Flutter アプリ。

研究分野

  • 時系列予測・補完、スマートグリッド安定性向けの AI sLM LLM

  • 凍結 小型言語モデル（sLM）・LLM のスペクトル再プログラミング

  • 分散型エネルギー資源向けの IoT およびエッジ知能

  • 電力ネットワーク トポロジーへのグラフ ニューラル ネットワーク適用

主な実績

  • CVFlow（cvflow.app）と AIReco を提供。産業向け IoT の SensoryHub にコントリビューター。

  • スタートアップ・産業（D'isum）・Anohou Solutions にわたり、5 年以上フルスタックおよびバックエンド製品を構築。
